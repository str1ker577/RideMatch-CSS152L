What's New with RideMatch!

For our CSS152L group project, we are continuing the development of RideMatch — now introducing a wave of exciting new features and improvements based on your feedback and evolving user needs.

Patch 1.0.1 - Updated Login/Sign up feature

- Fixed issue where users couldn't login even with valid credentials.

- Added a user profile icon next to the the Navigation Bar's top right, this way users can toggle to sign out of their accounts.

- Improved backend error handling for smoother user authentication.

- Added a login button to the top right of the naviation for easier access.

- The User Icon in the sidebar now functions contextually.

  • • When logged in, it takes you to your profile page.
  • • If not logged in, it prompts you to login or sign up.
May 31st, 2025

Patch 1.0.2 New Pages & Layout Foundation

- Introduced several new core pages as part of RideMatch's expanded functionality:

  • • Forum - A dedicated space for users to ask questions or share car-related discussions.
  • • Patches/What's New - A changelog-style page to keep users informed about recent updates.
  • • Profile - Personlized profile pages for each registered user.

- All new pages use a consistent layout for a seamless browsing experience.

- Updated the side navigation menu to include links to the newly added pages, making navigation easier and more intuitive.

June 1st, 2025

Patch 1.0.3 Added Fully Electric Option and Added Electric Cars to the database

- Introduced a new “Fully Electric” filter option in the car search function. Expanded the vehicle database to include electric car specifications.

June 6th, 2025

Patch 1.0.4 Added Calculator Page and Calculator Icon to the sidebar

- Created a new Calculator.html and added the default header and footer layout.

- Added a new Icon to the side bar menu that takes you to the calculator page.

June 6th, 2025

Patch 1.0.5 Added Sorting Option in Results Table

- Implemented a dropdown filter in the results table header using a filter icon.

  • • Users can now sort by Price (Low to High / high to Low).
  • • Users can now sort by Horsepower (Low to High / high to Low).
  • • Users can now sort by Cargo Space (Low to High / high to Low).
  • • Users can now sort by Price (Low to High / high to Low).
